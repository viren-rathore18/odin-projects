Etch-a-Sketch

Create pixel art by hovering over the grid. Customize your canvas size and unleash your creativity!

Drawing Canvas

How to Use

  • Enter a grid size (1-100) and click Generate to create your canvas
  • Hover over squares to draw - each pass adds color
  • Use Reset to clear your current drawing
  • Use New Grid to create a fresh canvas with your current size
  • Try different grid sizes for finer or coarser detail
